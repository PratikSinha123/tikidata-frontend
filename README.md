# ⚽ Tiki Data — Professional Football Analytics

![Tiki Data Logo](GUI/src/main/resources/static/img/logo.png)

Tiki Data is a premium, high-performance football analytics dashboard designed to provide deep insights into match statistics, player performance, and league standings. Built with a modern Java Spring Boot backend and a sleek, dynamic frontend, it offers a professional-grade experience for football enthusiasts and analysts.

## 🚀 Features

- **Live Match Dashboard**: Real-time visualization of ongoing and historical matches.
- **Advanced Standings**: Dynamic league tables with detailed performance metrics.
- **Deep Analytics**: Comprehensive player stats, lineups, and match event tracking.
- **Clean UI**: A modern, dark-themed interface inspired by elite platforms like Sofascore.
- **Data-Driven**: Efficient processing of CSV-based football data archives.

## 🛠️ Technology Stack

- **Backend**: Java Spring Boot
- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Data Management**: File-based CSV processing for rapid data ingestion.
- **DevOps**: Git-ready structure for seamless deployment.

## 📦 Getting Started

To run the application locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/hunterbrotha/Java-Project.git
   ```
2. Navigate to the GUI folder and run the bootstrapper:
   ```powershell
   cd GUI
   .\start.bat
   ```
3. Open your browser and visit: `http://localhost:8082`

## 📂 Project Structure

- `GUI/`: The main Spring Boot application and frontend assets.
- `scripts/`: System automation and run scripts.
- `TikiDataProject.zip`: A clean, portable distribution of the project.
